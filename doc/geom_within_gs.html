geom_within_gsR Documentation

Test if a resource (HP) geometry is within a Grid Square (gs).

Description

Test if the geometry of a resource (eg. Heritage Place) is within a Grid Square (gs). If so, return the ID of the Grid Square.

Usage

geom_within_gs(
  resource.wkt = NA,
  gs.path = paste0(system.file(package = "eamenaR"), "/extdata/grid_squares.geojson"),
  verbose = T
)

Arguments

resource.wkt

the WKT geometry of a resource, as a character format. This WKT geometry can comes from a BU sheet (ex: "POINT(0.9 35.8)").

gs.path

a path to a GeoJSON file, by default to the example 'grid_squares.geojson' This GeoJSON is an EAMENA output of the Grids as a GeoJSON URL. The name of the grids can be calculated with the geom_bbox function. The GeoJSON is read and convert to a 'sf' object.

verbose

if TRUE (by default): verbose.

Details

when this function is called for a dataframe, it has to be nested into a loop.

Value

the ID of the Grid Square for each WKT geometries.

Examples


library(dplyr)

geom_within_gs(resource.wkt = "POINT(0.9 35.8)")
