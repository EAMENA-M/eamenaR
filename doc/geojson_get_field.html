geojson_get_fieldR Documentation

Get values of a given field in a GeoJSON file.

Description

Get values of a given field

Usage

geojson_get_field(geojson.path, field = "EAMENA.ID")

Arguments

geojson.path

the path to the GeoJSON file, eg: "https://raw.githubusercontent.com/eamena-oxford/eamena-arches-dev/main/data/geojson/caravanserail.geojson"

field

a field name, in R format, eg: EAMENA.ID

Value

A vector with all values

Examples


geojson_get_field(geojson.path = "https://raw.githubusercontent.com/eamena-oxford/eamena-arches-dev/main/data/geojson/caravanserail.geojson")
