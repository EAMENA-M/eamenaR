geom_bboxR Documentation

Create the bounding box of BUs' geometries and save it as a GeoJSON file

Description

Read one or various XLSX worksheets within a folder. Collect the xmin, xmax, ymin, xmax and creates the bounding box, that is to say, the extent of the geometries. Can be used to preselect grids, to get stamen (ggmap), etc. Used before the function geom_within_gs and list_mapping_bu.R to get the IDs of the potential grids.

Usage

geom_bbox(
  dataDir = NA,
  x_column = "xcoord",
  y_column = "ycoord",
  wkt_column = NA,
  dirOut = dataDir,
  geojson.name = "geojson.geojson"
)

Arguments

dataDir

the path to the folder where the XLSXs are

x_column, y_column

the column of the X and Y coordinates, if NA the function will look the field 'wkt'.

wkt_column

the column of the WKT coordinates. Useful if the X,Y coordinates are already in

dirOut

the path to the folder where the GeoJSON file will be created.

geojson.name

the name of the GeoJSON that will be created.

Value

a GeoJSON file

Examples


geom_bbox(dataDir = "C:/Rprojects/eamena-arches-dev/data/bulk/bu/")
